수신자 코드

1. 기존의 fifo 파일이 존재하면 삭제 unlink()
2. 새로운 FIFO 파일 생성
3. 이를 읽기 전용으로 열기
4. FIFO로부터 데이터를 읽고 화면에 출력
5. 파일 닫기

송신자 코드
1. fifo 파일을 쓰기 전용으로 열기
2. stdin(0) 으로부터 데이터를 읽음
3. FIFO에 데이터를 씀
4. 파일 닫기

실행 순서 및 결과
터미널 1에서 reader 프로그램 실행(데이터 수신 대기)
터미널 2dptj writer 프로그램 실행
writer에서 키보드로 입력한 데이터가 FIFO를 통해 reader에게 전달되어 화면에 출력
